# Kanban Board

A simple Kanban board built with **Vue 3**, **Vuetify**, **TypeScript**, and the **Composition API**. It allows users to:

- Add new task cards
- Edit existing cards
- Delete cards
- Drag and drop cards between status columns (To Do, In Progress, Done)

The project is deployed on **Vercel**.
